
## Описание

Серверная часть приложения для игры в шахматы. Фреймворк NestJS.

Новое лобби для игры создается через Get запрос, остальное общение с клиентом через Websocket.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
